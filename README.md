# DryDock AI — Website

Marketing site for DryDock (usedrydock.com): governance for the AI delivery era.

- `index.html` — the complete site (single file: HTML, CSS, JS)
- `drydock-logo-*.svg` — brand assets (app tile, dark mark, light mark)

## Deploy
Drag the folder into Netlify (Deploys → drag & drop) or serve `index.html` from any static host.
